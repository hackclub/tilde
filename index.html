You've stumbled across the Hack Club Tilde server!

What's a 'Tilde server'?

$ man 7 tilde

The Hack Club Tilde is a quasi-public Linux server offering free access to anyone in Hack Club.

Stuff you can do on this includes:

  • Host a website
  • Try out gemini
  • Learn how to use the shell
  • Try a new programming language
  • Chat with hackers
  • All of the above!

How can I get access?

Easy! Just join #tilde on hackclub.slack.com and ping an admin!

(You'll need a valid SSH key)

What's on the server?

There's web hosting, a local irc server, local e-mail (doesn't leave the server, for right now), and tons of programming languages

Missing something? Hop on irc or contact Madeline on slack!
